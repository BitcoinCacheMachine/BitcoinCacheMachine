

Description:
  Get and set BCM CLI configuration options. Also see 'bcm info'.

Usage:
  bcm config [command]

Commands:
  get      Get the value of a BCM configuration option.
  set      Set the BCM CLI configuration option.

Configuration Options:
  chain=<active_chain>      Get or set the target bitcoin blockchain (mainnet, testnet, regtest).
  runtime-dir=<runtime_dir> Set the directory in which BCM keeps all sensitive client-side info (default '~/.bcm').

  Use "bcm config [command]" to get help for that command.
