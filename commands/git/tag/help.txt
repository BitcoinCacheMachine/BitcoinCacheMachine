Description:
  Tag a git commit backed by your Trezor.

Usage:
  bcm git tag --tag="v0.0.1" --message="Version 1"

Parameters:
  --tag=                      The name of the tag. Typically indicated in version numbers.
  --message=                  The annotation for the tag.
  
Optional Flags:
  --verify                   Verify tag signature
  --delete                   Delete tags (local)
